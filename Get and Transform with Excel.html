Get and Transform with Excel

In this mini project, I assumed the role of a data analyst for a cosmetic company - ABZ Cosmetic Group.
I am expected to perform some analysis on the Call and Collect [Sales] Data generated from stores in Australia for the next periodic review.

Business Question

Management wants to know:
- How much of sales is generated by the various brands,
- How much of sales is generated by the various chains across Australia,
- Top 5 sales agents for the period ended.

Nature and Sources of Data

- Agent Information and Product list are available within the company's systems,
- Call and Collect [CC] data is received on a weekly basis in text files and saved in the CallCollect folder.

Actions

- Converted all Excel data sets to Tables where necessary,
- Created a query [Agents] that returned agents full names in a new column on the Agent List Table,
- Created a query [Product List] for all products in sale,
- Created a query [CallCollect] that picked up all the Call and Collect Data files from the CallCollect folder, combined all weekly records and formated all columns to the appropriate data type,
- Creates a new query [Stores] comprising all 3 queries representing the different stores to serve as a central source for all Store Information,
- Created a final query [Transaction Lines] which is a merge of all relevant data from the CallCollect, Stores, and Agents queries created above.

- This final query - Transaction Lines - comprised of:
* All data from CallCollect Query,
* Store Name and Chain Name from the Stores Query via an Inner Join merge-type,
* Agent Name from the Agents Query via a Left Outer Join merge-type, and
* Brand Name from the Product List Query via an Inner Join merge-type.

- Loaded the Final Query - Transaction Lines - to a new sheet; Call & Collect, for all final analysis with PivotTables.
